#Required
## Requirements

### Open the application
[Open Coolha DApps↗](https://coolha.com)

### Blockchain Wallet
No wallet? [Select wallet↗](https://ethereum.org/wallets/find-wallet/)

### Blockchain account
This is the prerequisite for entering the web3 world, but currently there are `smart accounts`, which use `account abstraction` to realize mobile phone numbers, email addresses, Google accounts, Apple accounts, etc. to control accounts on the blockchain, so that users do not need to learn professional areas Blockchain knowledge can complete on-chain operations.

### Lens Account
This is the core of realizing decentralized social networking. Each Lens account is an independent `smart account', controlled through the user's blockchain account and off-chain account. [How to register? ](/start/signup.md)
