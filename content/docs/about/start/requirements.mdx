# 必备条件

### 打开应用
[Open Coolha DApps↗](https://coolha.com)

### 区块链钱包
没有钱包？[选择钱包↗](https://ethereum.org/wallets/find-wallet/) 

### 区块链账户
这是进入 web3 世界的前提，但目前有`智能账户`，通过`账户抽象`实现手机号码、电子邮箱、Google账户、Apple账户等等控制区块链上的账户，让用户不需要学习专业区块链知识即可完成链上操作。

### Lens账户
这是实现去中心化社交的核心，每个Lens账户都是一个独立的`智能账户`，通过用户的区块链账户和链下账户控制。[如何注册？](/docs/start/signup)
