# 介绍 Coolha Community

Coolha Community 是一个开放、包容的去中心化社区，欢迎所有对 Web3 社交、创作者经济和去中心化治理感兴趣的用户参与。

## 社区特色

- **开放治理**: 通过 DAO 机制参与社区决策
- **代码贡献**: 为智能合约和前端应用贡献代码
- **多语言支持**: 帮助翻译文档和界面
- **功能建议**: 提出新功能和改进建议
