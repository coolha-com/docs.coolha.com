# 帮助翻译

帮助我们将 Coolha 带给全世界的用户！我们需要社区的帮助来翻译文档、界面和其他内容。

## 翻译项目

### 为 coolha.com 翻译

主网站的翻译包括：

- **首页内容**: 产品介绍和功能说明
- **用户界面**: 按钮、菜单、表单等 UI 元素
- **帮助文档**: 用户指南和常见问题
- **营销内容**: 博客文章和公告

**支持语言**:
- 英语 (English)
- 中文 (简体/繁体)
- 日语 (日本語)
- 韩语 (한국어)
- 阿拉伯语 (العربية)

### 为 docs.coolha.com 翻译

文档网站的翻译包括：

- **技术文档**: API 文档和开发指南
- **用户手册**: 使用教程和操作指南
- **概念解释**: Web3 和区块链概念
- **最佳实践**: 安全指南和建议


## 翻译流程

### 1. 加入翻译团队

1. **申请加入**: 联系我们，发送翻译申请
2. **语言能力**: 说明您的语言能力和翻译经验
3. **获得权限**: 获得翻译平台的访问权限
4. **开始翻译**: 选择您想要翻译的内容

### 2. 翻译工具

我们使用以下工具进行翻译管理：

- **GitHub**: 文档翻译的版本控制
