# 帮助翻译

帮助我们将 Coolha 带给全世界的用户！我们需要社区的帮助来翻译文档、界面和其他内容。

## 翻译项目

### 为 coolha.com 翻译

主网站的翻译包括：

- **首页内容**: 产品介绍和功能说明
- **用户界面**: 按钮、菜单、表单等 UI 元素
- **帮助文档**: 用户指南和常见问题
- **营销内容**: 博客文章和公告

**支持语言**:
- 英语 (English)
- 中文 (简体/繁体)
- 日语 (日本語)
- 韩语 (한국어)
- 西班牙语 (Español)
- 法语 (Français)
- 德语 (Deutsch)

### 为 docs.coolha.com 翻译

文档网站的翻译包括：

- **技术文档**: API 文档和开发指南
- **用户手册**: 使用教程和操作指南
- **概念解释**: Web3 和区块链概念
- **最佳实践**: 安全指南和建议

**文档结构**:
- 开始使用
- 商业功能
- 创作者工具
- 社区参与
- API 参考

## 翻译流程

### 1. 加入翻译团队

1. **申请加入**: 发送邮件到 [translate@coolha.com](mailto:translate@coolha.com)
2. **语言能力**: 说明您的语言能力和翻译经验
3. **获得权限**: 获得翻译平台的访问权限
4. **开始翻译**: 选择您想要翻译的内容

### 2. 翻译工具

我们使用以下工具进行翻译管理：

- **Crowdin**: 主要的翻译管理平台
- **GitHub**: 文档翻译的版本控制
- **Figma**: UI 界面翻译的设计稿
- **Notion**: 翻译进度和协调

### 3. 翻译规范

#### 术语一致性

保持关键术语的一致性：

| 英文 | 中文 | 日语 | 韩语 |
|------|------|------|------|
| DApp | 去中心化应用 | DApp | 디앱 |
| Wallet | 钱包 | ウォレット | 지갑 |
| NFT | 非同质化代币 | NFT | NFT |
| DAO | 去中心化自治组织 | DAO | DAO |

#### 语言风格

- **简洁明了**: 使用简单易懂的语言
- **本地化**: 符合目标语言的表达习惯
- **专业准确**: 技术术语翻译准确
- **用户友好**: 面向普通用户的友好表达

### 4. 质量控制

- **同行评审**: 其他翻译者审核您的翻译
- **原生审校**: 母语使用者进行最终审校
- **测试验证**: 在实际界面中测试翻译效果
- **用户反馈**: 收集用户对翻译质量的反馈

## 翻译指南

### Web3 术语翻译

常见 Web3 术语的翻译建议：

- **Smart Contract**: 智能合约
- **Blockchain**: 区块链
- **Cryptocurrency**: 加密货币
- **Decentralized**: 去中心化
- **Token**: 代币
- **Mining**: 挖矿
- **Staking**: 质押
- **Governance**: 治理

### 用户界面翻译

界面翻译的注意事项：

- **长度限制**: 考虑不同语言的文本长度差异
- **文化适应**: 适应目标文化的使用习惯
- **可读性**: 确保在小屏幕上也能清晰阅读
- **一致性**: 保持整个应用的术语一致

### 文档翻译

文档翻译的要求：

- **结构完整**: 保持原文的结构和格式
- **链接有效**: 确保翻译后的链接仍然有效
- **代码示例**: 代码注释可以翻译，代码本身保持不变
- **图片说明**: 翻译图片中的文字说明
