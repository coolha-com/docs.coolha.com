# 代码贡献

欢迎开发者为 Coolha 项目贡献代码！我们的项目是开源的，欢迎各种形式的技术贡献。

## 项目结构

### 智能合约

Coolha 的智能合约基于以太坊和 Lens Protocol 构建：

- **核心合约**: 用户身份和社交图谱
- **内容合约**: 帖子、评论、点赞等功能
- **经济合约**: 代币经济和激励机制
- **治理合约**: DAO 治理和投票

#### 先进技术支持

我们使用最新的区块链技术：

- **Solidity 0.8+**: 最新版本的智能合约语言
- **OpenZeppelin**: 安全的合约库
- **Hardhat**: 开发和测试框架
- **Layer 2**: 支持 Polygon 等二层网络

#### 漏洞排查

安全是我们的首要任务：

- **代码审计**: 定期进行安全审计
- **漏洞赏金**: 提供漏洞发现奖励
- **测试覆盖**: 保持高测试覆盖率
- **形式化验证**: 关键合约的数学验证

### 前端网站

我们的前端项目使用现代 Web 技术栈：

#### 应用网站开发

主要技术栈：

- **React/Next.js**: 现代前端框架
- **TypeScript**: 类型安全的 JavaScript
- **Tailwind CSS**: 实用优先的 CSS 框架
- **Web3 库**: ethers.js, wagmi 等

功能模块：

- **用户界面**: 响应式设计和用户体验
- **钱包集成**: 支持多种 Web3 钱包
- **内容管理**: 帖子创建、编辑、展示
- **社交功能**: 关注、点赞、评论、分享

#### 文档网站开发

文档技术栈：

- **Fumadocs**: 现代文档框架
- **MDX**: Markdown + React 组件
- **多语言**: 国际化支持
- **搜索功能**: 全文搜索和导航

## 贡献流程

### 1. 准备工作

```bash
# 克隆仓库
git clone https://github.com/coolha/coolha-dapp.git
cd coolha-dapp

# 安装依赖
npm install

# 设置环境变量
cp .env.example .env.local
```

### 2. 开发环境

```bash
# 启动本地开发服务器
npm run dev

# 运行测试
npm run test

# 代码检查
npm run lint
```

### 3. 提交代码

1. **创建分支**: `git checkout -b feature/your-feature`
2. **编写代码**: 遵循代码规范
3. **添加测试**: 确保功能正常
4. **提交更改**: 使用清晰的提交信息
5. **创建 PR**: 提交 Pull Request

## 代码规范

### 智能合约

- 遵循 [Solidity Style Guide](https://docs.soliditylang.org/en/latest/style-guide.html)
- 使用 NatSpec 注释
- 编写全面的测试
- 进行 gas 优化

### 前端代码

- 使用 ESLint 和 Prettier
- 遵循 React 最佳实践
- 编写 TypeScript 类型定义
- 添加单元测试和集成测试
